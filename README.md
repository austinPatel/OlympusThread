# Olympus Technical Assessment
# Technical Notes

## Overview
 This Application is developed in C# (.NET 8) console application
• Multithreading implemented using Task
• Thread safety using:
   - Interlocked.Increment() for auto counter updates
   - lock() for synchronized file access
• OOP Concept which is implemented using:
   - Models
   - Services
   - Utility classes for the date time format.
• Docker container created using Linux runtime
• File path /log/out.txt mapped to host machine using Docker volume

## Docker Build Command
docker build -t olympus-thread .

## Docker Run Command
docker run -i -v c:\junk:/log olympus-thread
